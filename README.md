# breadboard tutorial yayay






> Built in [Breadboard](https://breadboard.hackclub.com), a Hack Club program. This project took ~1.8 hours of work.



## What It Does

A knob sets how bright the LED glows and the button turns it on and off - when it's switched back on, it remembers the brightness setting.


## How It Works

The circuit is captured in `breadboard-project.json`, and the firmware that runs it is in the `firmware/` folder.


## How To Use It

Press the button and turn the knob!
